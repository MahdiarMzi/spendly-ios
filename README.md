
# Spendly 💰

AI-powered personal finance management app built with SwiftUI.

Spendly helps users track expenses, manage income, monitor recurring payments, analyze spending habits, and gain actionable financial insights through a modern and intuitive mobile experience.

## Features

### Expense Tracking

- Add, edit, and delete expenses

- Categorize transactions

- Track spending history

- Search and filter expenses

### Income Management

- Record income sources

- Track monthly earnings

- Monitor cash flow

### Recurring Payments

- Manage subscriptions

- Track recurring expenses

- Payment reminders

### Analytics & Insights

- Spending statistics

- Category breakdowns

- Financial health monitoring

- Expense trends

### AI Features

- Receipt scanning

- Bank statement analysis

- Spending anomaly detection

- AI-powered financial insights

- Smart recommendations

## Tech Stack

- Swift

- SwiftUI

- MVVM Architecture

- AI Integration

- iOS Native Development

## Development Process

This project was developed with AI-assisted development tools for brainstorming, implementation support, debugging, and code review.

Final architecture, customization, integration, testing, and project decisions were performed and validated by the developer.
